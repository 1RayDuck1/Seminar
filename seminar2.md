## Branch creating 

* *git branch branch_name* - Команда создающая отдельную ветку.
* *git checkout* - Команда для перехода между коммитами или ветками.
* *git branch -d branch_name* - Удалить ветку.

## Branch merging

* *git merge branch_name* - команда слияющая ветки.
